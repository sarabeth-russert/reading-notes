## Code 301
#### Day 11

**EJS Partials**

>"think of partials as functions"

- set up a directory called partials within views
- create a file within partials
- put html markup want to be able to reuse in the file
- go to the ejs file you want to paste your partial into
- `<%- include( PARTIAL_FILE ) %>` where you want to paste it

So easy!
